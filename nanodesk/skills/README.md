# Nanodesk Skills

存放你的个人技能（SKILL.md 文件）。

## 什么是 Skill

Skill 是 nanobot 的指令集，通过 Markdown 文件定义 AI 的行为模式。

## 示例

```markdown
# My Workflow

## 任务：处理每日报告

1. 读取 ~/reports/daily/ 下的文件
2. 总结关键信息
3. 发送到钉钉
```

## 使用方法

将 `.md` 文件放在此目录，nanobot 会自动加载。

更多请参考 [nanobot skills 文档](../nanobot/skills/README.md)
