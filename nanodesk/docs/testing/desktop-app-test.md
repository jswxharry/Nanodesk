## 手动测试

> 👤 **人类执行**

### 1. 启动测试

**步骤**:
1. 运行 `.\nanodesk\scripts\dev\run_desktop.ps1`
2. 观察窗口标题（应为 "Nanodesk Desktop vx.x.x"）
3. 观察状态栏（应显示"就绪"）

**预期结果**: 正常启动，无异常

---

### 2. Gateway 启停

**步骤**:
1. 点击"启动 Gateway"
2. 等待 5 秒，观察状态栏
3. 点击"停止 Gateway"
4. 观察状态栏

**预期结果**: 启停正常，无崩溃

---

### 3. 系统托盘

**步骤**:
1. 启动 Gateway
2. 最小化到托盘
3. 右键托盘图标，测试菜单项
4. 点击"退出"

**预期结果**: 托盘功能正常，退出无崩溃
