# 飞书集成测试用例

## 测试内容

飞书机器人基础对话、多轮对话、Markdown 卡片

---

## 测试步骤

### 1. 基础对话测试

```powershell
# 前置：确保 config.json 中飞书配置正确

# 步骤
1. 启动 Gateway
2. 在飞书群中 @机器人
3. 发送："你好"
4. 观察机器人是否回复
```

**预期结果**：机器人正常回复，无异常

---

### 2. 多轮对话测试

```powershell
# 步骤
1. 第1轮："我叫张三"
2. 第2轮："我叫什么名字？"
3. 观察机器人是否记得上下文
```

**预期结果**：能正确回答"张三"

---

### 3. Markdown 卡片测试

```powershell
# 步骤
1. 让 Agent 执行带表格的结果（如搜索）
2. 观察飞书消息格式
3. 检查表格是否正常显示
```

**预期结果**：表格格式正确，无错乱

---

### 4. 长时间运行测试

```powershell
# 步骤
1. Gateway 运行 1 小时
2. 期间不定期发送消息
3. 观察是否稳定回复
```

**预期结果**：长时间运行稳定，无断连
