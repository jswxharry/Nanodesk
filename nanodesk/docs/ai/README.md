# AI 开发文档

本目录包含供 AI 助手（如 Kimi Claw AI）开发 Nanodesk 时参考的文档。

## 目的

- 记录 AI 开发 Nanodesk 的最佳实践
- 提供项目结构和开发流程的快速参考
- 减少 AI 每次开发时的上下文重建成本

## 文件说明

| 文件 | 用途 |
|------|------|
| `KIMI_CLAW_AI_GUIDE.md` | 完整的开发指导手册，包含项目结构、工作流、配置管理、测试规范等 |

## 使用方式

**对于 AI 开发者：**

1. 首次参与 Nanodesk 开发时，先阅读 `KIMI_CLAW_AI_GUIDE.md`
2. 根据指导文件了解项目结构和开发模式
3. 参考文档中的"常见任务"快速上手

**对于人类开发者：**

- 本目录主要服务于 AI 协作开发
- 如需用户文档，请查看 `docs/setup/`
- 如需设计文档，请查看 `docs/design/`

## 更新维护

- 当项目结构变化时，同步更新 `KIMI_CLAW_AI_GUIDE.md`
- 当新增开发模式或工具时，补充到指导文档
- 定期回顾文档，移除过时内容
